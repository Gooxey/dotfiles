# dotfiles

All files are stored like they are in $HOME ( with '.'-prefixes removed )

## Installation

> **Note**: Running the script will delete all files in the directories
> listed by the script. ( When executing the script, you will have a confirmation screen )

> **Note**: This script will create symbolic links to the files inside this repo. Therefore,
> make sure to not delete this directory after running the install script.

Run the install.sh script:

```bash
bash install.sh
```
